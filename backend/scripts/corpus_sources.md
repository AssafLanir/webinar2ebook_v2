# Corpus Sources Registry

Track public transcript sources for the calibration corpus.

## Source Template

Copy this block for each transcript:

```
### [ID] Title
- **URL**:
- **Bucket**: (1-10, see real_corpus_eval.md)
- **Category**: (e.g., Product launch, Panel, Q&A-heavy)
- **Stressors**:
  - CTA density: Low / Medium / High
  - Acronym density: Low / Medium / High
  - # Speakers:
  - Q&A %:
  - ASR quality: Clean / Some noise / Messy
- **License/Terms**: (e.g., YouTube Standard, CC-BY, Unknown)
- **Notes**:
- **Local file**: corpora/private/XX_short_name.md
```
